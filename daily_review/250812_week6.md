## Problem (개선할 점)
### 1. 회의 집중력 저하
- 개인 작업할 때는 집중력이 나름 좋았는데 API 논의를 할 때는 상대적으로 집중력이 떨어졌다. 
- 디버깅 해결을 못해서 그런지 집중이 잘 안되었다. 
- 팀원들에게 미안했다. 

## Try (해결책)
- 회의할 땐 회의만, 개인작업할 때는 개인작업에만 집중하자. 
- 전환이 필요할 땐 잠깐 휴식을 취하고 하자.
